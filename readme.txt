Install WPL packages on your Real Estate website.

== Description ==

This plugin connects your WPL website to Realtyna repository to install packages. You can debug your website or configure it using the pre-defined packages that we created very fast and simple.

To use this plugin you need [WPL Real Estate](https://wordpress.org/plugins/real-estate-listing-realtyna-wpl/) installed and activated.

== Installation ==

For installing Realtyna Provisioning please follow below steps:

1. Install it either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate the plugin through the 'Plugins' menu in WordPress.
